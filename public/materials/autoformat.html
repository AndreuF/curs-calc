Formatació automàtica

L'OpenOffice.org Calc permet aplicar a un rang de cel·les, que prèviament s'ha seleccionat, una sèrie de formats preestablerts que faciliten la tasca de donar format a una part del full de càlcul. Per utilitzar aquesta opció

  • Demaneu l'opció de menú Format | Formatació automàtica…

  • El botó Més permet tenir accés als elements de format que s'aplicaran (activats) o que no s'han d'aplicar (desactivats)

Elements de la finestra

  • Format
    Permet escollir el model de formatació automàtica, dels formats preestablerts, a aplicar a la zona seleccionada.
  • Requadre de vista prèvia
    Mostra el resultat de formatació que tindria la zona seleccionada en aplicar el format i les opcions escollides.
  • D'acord
    Aplica el format automàtic a la selecció i tanca el diàleg.
  • Cancel·la
    Tanca la finestra de formatació automàtica sense aplicar canvis.
  • Afegeix
    Permet crear un nou model de format preestablert basat en el que tingui l'àrea actual seleccionada (que ha de tenir un mínim de 4 x 4 cel·les). Així es poden crear nous formats automàtics personalitzats.
  • Suprimir
    Permet eliminar de la llista l'element de format assenyalat
  • Mes
    Mostra o amaga les opcions de formatació a aplicar
    • Formatació
      Permet determinar quins elements,del model de format seleccionat, s'aplicaran o no a l'àrea seleccionada del full
  • Reanomena
    Permet modificar el nom del format automàtic seleccionat.
